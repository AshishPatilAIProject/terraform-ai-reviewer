Remediation Plan

Finding: S3 Bucket Missing Encryption
Risk: Data stored in unencrypted buckets is vulnerable to unauthorized access and data breaches.
Terraform Fix:
```hcl
resource "aws_s3_bucket" "example" {
bucket = "my-bucket"
server_side_encryption_configuration {
rule {
apply_server_side_encryption_by_default {
sse_algorithm = "AES256"
}
}
}
}
```

Finding: Security Group Allows Open SSH Access
Risk: Allowing SSH access from 0.0.0.0/0 exposes resources to brute force attacks and unauthorized access.
Terraform Fix:
```hcl
resource "aws_security_group" "ssh" {
name = "restrict-ssh"

ingress {
from_port = 22
to_port = 22
protocol = "tcp"
cidr_blocks = ["203.0.113.0/24"] # Restrict to trusted IP range
}
}
```

Finding: S3 Bucket Without Versioning
Risk: Lack of versioning increases risk of permanent data loss from accidental deletion or overwrites.
Terraform Fix:
```hcl
resource "aws_s3_bucket_versioning" "example" {
bucket = aws_s3_bucket.example.id
versioning_configuration {
status = "Enabled"
}
}
```

Finding: S3 Bucket Without Public Access Block

resource "aws_s3_bucket_public_access_block" "example" {

}
